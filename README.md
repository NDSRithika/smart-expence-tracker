# 💰 Ledger – Smart Personal Finance Advisor

A modern **Smart Personal Finance Advisor** built using **Python (Flask), HTML, CSS, JavaScript, and SQLite**. The application helps users manage their income, expenses, monthly budgets, and savings goals while providing intelligent spending analysis and financial recommendations.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- Secure User Login
- Password Hashing
- Session Management
- Logout

### 💵 Expense Management
- Add Expenses
- Edit Expenses
- Delete Expenses
- Search Expenses
- Filter by Category
- Monthly Expense Tracking

### 💼 Budget Planning
- Enter Monthly Income
- Set Monthly Savings Goal
- Calculate Available Spending Budget
- Budget Progress Tracking
- Remaining Budget Calculation

### 📊 Financial Analysis
- Category-wise Expense Analysis
- Smart Spending Suggestions
- Budget Health Score
- Savings Prediction
- Monthly Financial Summary

### 📈 Charts & Reports
- Pie Chart (Category-wise Spending)
- Bar Chart (Monthly Spending)
- Line Chart (Expense Trends)

### 🎨 User Interface
- Responsive Design
- Modern Dashboard
- Dark/Light Theme
- Interactive Charts
- Clean and User-Friendly Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Expense-Tracker.git
```

### Move into the Project Folder

```bash
cd Smart-Expense-Tracker
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:3000
```
